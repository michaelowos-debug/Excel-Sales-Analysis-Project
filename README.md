# Excel Sales Analysis Project

## Project Overview
This project focuses on analyzing sales data using Microsoft Excel to uncover trends, patterns, and business insights.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Dashboards

## Key Tasks Performed
- Data cleaning and formatting
- Creating pivot tables
- Building interactive charts
- Analyzing sales performance by region, product, and time

## Insights Generated
- Identified top-performing products
- Compared sales across regions
- Observed monthly and yearly trends
